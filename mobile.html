Mobile Web Course

A responsive course website

Overview

Currently, there is no official website for the CMU course 67-240, Mobile Web Design. The goal of this project was to design an accessible and responsive website that provides information about 67-240 to a diverse audience consisting of students currently in the course, planning to be in the course, potential clients, and potential guest speakers.

Duration
Spring 2020 (2 months)

Team
Chris Haas, Lily Huan, Amy Luo

Skills
UX Design, User Testing

Process

Competitive Analysis

To begin designing the site, our team first looked into other course or project showcase websites were designed and assessed the patterns in strengths and weaknesses of a few examples in order to get inspiration for our course site development.

    Strengths:
  • clean layout, good use of white space
  • easy navigation with side navbar
    Weaknesses:
  • lack of responsiveness (not mobile compatible)
  • elack of graphics and images
Target Audience

Reflections

Overall, this project was a great learning opportunity, as we were able to gain real world experience working through the iterative design process collaboratively. Halfway through the project, we also had to quickly adapt to the challenges of remote work due to the course shifting online. In the end, we were able to maintain the same project vision throughout the process and produced an easy-to-navigate and responsive course website.
