CLOUD CAT

A two-player platformer game

Overview

For my Fundamentals of Programming course, I created a multiplayer platformer game. The player is a cat that gains points by jumping from cloud to cloud and killing attacking enemies for survival. The objective is to travel through the sky as far as possible without losing all three lives. This project was programmed with Python using the pygame module.

Duration
Fall 2018 (2 weeks)

Team
Individual Project

Skills
Python programming, game design

Process

Storyboarding

To first brainstorm an idea of the game mechanics and playthrough, I created rough storyboards of how I envisioned the game. Some aspects of this inital idea were changed such as making the game set in a cloud environment, where the player cannot drop down to the ground, for an added level of difficulty.

Gameplay

  • Controls: arrow keys to move, space to shoot (Player 1) and AWSD to move, ctrl to shoot (Player 2)
  • FeaturesEach character has 3 lives. As players advance, they level up and difficulty increases
  • Dark Clouds: drop rain and disappear after being jumped on
  • Bees: shoot bullets and move up and down
  • Power ups: point boost with coins and gain extra life with hearts
  • Level Editor: customize cloud platform pattern
